# PRD 06 — Budget & Cashflow (Fintech)

## 1. Overview

Application to analyze income, expenses, and cash flow based on transactions.

## 2. Product Objective

Help individuals or small businesses understand their financial situation.

## 3. Target Users

* Freelancers
* Small businesses

## 4. Main Use Cases

1. Import transactions
2. Categorize expenses
3. Visualize reports

## 5. Scope (In-Scope)

* CSV import
* Categorization rules
* Basic dashboards

## 6. Out of Scope

* Automatic bank integrations

## 7. Functional Requirements

1. Import transaction CSV files
2. Define categorization rules
3. Generate financial reports

## 8. Non-Functional Requirements

* Numerical accuracy

## 9. Inputs and Outputs

### Inputs

* CSV files

### Outputs

* Financial reports

## 10. Key Flows

Import → Categorize → Analyze

## 11. Known Edge Cases

* Malformed CSV files

## 12. Success Metrics

* Accurate financial totals

## 13. Assumptions and Constraints

* Single currency

## 14. Risks

* Parsing errors
